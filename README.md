# Craftbeerpi4 Mail Notifications Plugin

## Plugin to send Craftbeerpi4 Notifications via Mail

### Software installation:
Tested on Raspberry Pi OS Trixie

CBPi version 4.7.0

Please have a look at the [Craftbeerpi4 Documentation](https://openbrewing.gitbook.io/craftbeerpi4_support/readme/plugin-installation)

- Package name: cbpi4-mail-notifications
- Package link: https://github.com/eco37/cbpi4-mail-notofications/archive/refs/heads/main.zip

### Usage:

	- First Installation will add parameters to settings.
	- Configure settings.
	- Restart cbpi

### Parameters:

	- smtp_address: SMTP Address 
	- smtp_port: SMTP Port
	- smtp_user: Username to connect to SMTP Server
	- smtp_pass: Password to connect to SMTP Server
	- smtp_encryption: TLS, SSL or PLAIN(No encryption)
	- to_address: The address to recive the notifications
	- from_address: The Address the mail are sent from
